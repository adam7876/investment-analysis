# 美股投資分析系統 - 使用指南

## 🎯 系統概述

這是一個現代化的美股投資分析系統，提供直觀的Web介面，讓您無需使用終端機就能進行專業的投資分析。

## 🚀 快速開始

### 第一步：啟動系統

1. **打開終端機**（或命令提示字元）
2. **進入項目目錄**
   ```bash
   cd /Users/adammac/Documents/Investation/select
   ```
3. **啟動Web介面**
   ```bash
   python3 start_web.py
   ```
4. **等待啟動完成**
   看到以下訊息表示啟動成功：
   ```
   🌐 Web 介面啟動中...
   📱 請在瀏覽器中打開: http://localhost:5000
   ```

### 第二步：打開瀏覽器

1. **開啟您的瀏覽器**（Chrome、Safari、Firefox等）
2. **輸入網址**：`http://localhost:5000`
3. **開始使用**！

## 🎮 詳細操作指南

### 進行市場分析

1. **方法一：從首頁開始**
   - 點擊「快速分析」
   - 等待分析完成
   - 自動跳轉到儀表板查看結果

2. **方法二：直接使用儀表板**
   - 點擊導航欄「儀表板」
   - 點擊「重新整理」獲取最新數據
   - 查看分析結果

### 理解分析結果

**市場情緒指數：**
- 0-25：極度恐慌 → 考慮逢低買入
- 26-45：恐慌 → 謹慎樂觀
- 46-55：中性 → 持續觀察
- 56-75：貪婪 → 注意風險
- 76-100：極度貪婪 → 考慮獲利了結

**風險等級：**
- 🟢 Low：低風險，適合保守投資
- 🟡 Medium：中等風險，需要謹慎
- 🔴 High：高風險，建議減少部位

**投資建議：**
- 系統會根據當前市場環境提供具體建議
- 建議僅供參考，請結合個人情況決策

### 匯出和保存數據

1. **匯出報告**
   - 在儀表板點擊「匯出報告」
   - 系統會下載JSON格式的完整數據
   - 檔案名稱格式：`investment_analysis_YYYY-MM-DD.json`

2. **查看歷史數據**
   - 匯出的檔案包含完整的分析結果
   - 可以用文字編輯器或JSON查看器開啟

## 🔧 常見問題解決

### 問題1：無法啟動Web介面

**可能原因：**
- Flask未安裝
- 端口被占用
- Python版本不相容

**解決方法：**
```bash
# 安裝Flask
pip3 install Flask==3.0.0

# 檢查端口占用
lsof -i :5000

# 終止占用進程
kill -9 <PID>
```

### 問題2：頁面無法載入

**檢查步驟：**
1. 確認Web服務器正在運行
2. 檢查瀏覽器網址是否正確
3. 嘗試重新啟動服務器

### 問題3：數據載入失敗

**可能原因：**
- 網路連接問題
- API服務暫時不可用
- 系統正在處理中

**解決方法：**
1. 檢查網路連接
2. 等待幾分鐘後重試
3. 點擊「重新整理」按鈕

### 問題4：分析結果不準確

**說明：**
- 系統使用多個數據源
- 部分數據源可能暫時不可用
- 系統會使用可用數據進行分析

**建議：**
- 查看數據品質指標
- 結合多次分析結果
- 參考其他投資工具
